# GEPRC Child Theme

Thème enfant basé sur **Twenty Twenty-Four** pour reproduire l'esthétique sombre et néon de GEPRC avec une page d'accueil orientée produits et services FPV.

## Installation
1. Copier le dossier `geprc-child` dans `wp-content/themes` de votre site.
2. Vérifier que le thème parent `twentytwentyfour` est installé et actif sur le site.
3. Depuis l'admin WordPress, activer le thème **GEPRC Child**.
4. Attribuer la page "Accueil" au modèle **Front page (GEPRC Child)** et définir cette page comme page d'accueil statique.

## Personnalisation
- Dans **Apparence > Personnaliser**, section **Identité GEPRC**, ajustez les couleurs d'accent principales.
- La section produits se connecte automatiquement à WooCommerce : ajoutez des produits en vedette pour remplacer les blocs de démonstration.
- Le bouton "Personnaliser le thème" de la section services renvoie directement au Customizer.

## WooCommerce
- Le thème enfant déclare la compatibilité WooCommerce et applique une mise en forme sombre aux grilles produits.
- Si WooCommerce n'est pas installé, la grille affiche des cartes de remplacement sans générer d'erreur.
