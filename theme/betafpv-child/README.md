# BetaFPV Child Theme

Thème enfant inspiré du site BetaFPV, construit sur Twenty Twenty-Four pour l’éditeur de site. Il applique une palette sombre/bleu électrique, ajoute des blocs vitrine (hero, grille produits, FAQ) et reste compatible Beaver Builder Lite et WooCommerce.

## Installation
1. Copier le dossier `theme/betafpv-child` dans `wp-content/themes/` de votre instance WordPress.
2. Vérifier que le thème parent **Twenty Twenty-Four** est installé et actif.
3. Activer le thème **BetaFPV Child** depuis l’administration WordPress.

## Points clés
- Palette et typographies définies dans `theme.json` pour l’éditeur de site.
- Modèle `templates/front-page.html` prêt pour une page d’accueil vitrine (WooCommerce + vidéo + FAQ).
- Support WooCommerce et compatibilité Beaver Builder Lite (édition de pages/produits) dans `functions.php`.

## Personnalisation
- Modifiez les couleurs ou espacements dans `theme.json`.
- Remplacez l’image hero et les textes directement dans l’éditeur de site.
- Ajoutez vos produits WooCommerce pour alimenter la grille d’accueil (ou insérez vos propres blocs Beaver Builder).
