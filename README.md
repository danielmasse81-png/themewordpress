# themewordpress

Les thèmes enfants basés sur Twenty Twenty-Four sont disponibles dans le dossier `theme/` :

- `theme/geprc-child/` : palette sombre néon et page d’accueil vitrine pour GEPRC.
- `theme/betafpv-child/` : design inspiré de BetaFPV, prêt pour l’éditeur de site avec compatibilité WooCommerce et Beaver Builder Lite.

Copiez le dossier souhaité dans `wp-content/themes/` de votre instance WordPress, activez le thème enfant puis personnalisez depuis l’éditeur de site ou le customizer.
